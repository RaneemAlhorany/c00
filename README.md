# CPP Module 00

> The first C++ module from the 42 Common Core.
> This project introduces the fundamentals of C++98, classes, objects, member functions, namespaces, streams, initialization lists, and basic OOP concepts.

---

## About

CPP00 is the first project of the C++ modules at 42 School.

The goal is to move from procedural programming in C to Object-Oriented Programming using C++98.

During this module I learned:

- C++ compilation
- Namespaces
- Classes and Objects
- Member Functions
- Constructors
- Encapsulation
- stdio Streams
- Initialization
- Formatting Output
- C++98 compilation flags

## ⭐ Bonus

The bonus exercise extends the module by implementing an **Account Management System** that simulates banking operations using static class members.

### Concepts Practiced

- Static attributes
- Static member functions
- Object creation and destruction
- Constructors & Destructors
- Class-wide statistics
- Timestamp generation
- Formatted output
- C++98 coding standards

This exercise strengthened my understanding of how static data and member functions are shared between all instances of a class and how object lifecycle events can be tracked.
